"Perfect" frontend stack
------------------------

aim to create a good front-end stack with the following specs:
* Very good performance
* Use future proof technologies
* Fast build and good development workflow
* Easy to test

Included modules so far:
* React
* React Templates
* PageJS
* moment
* twitter bootstrap

Build tools:
* Gulp
* Rollup/Babel
* jshint
